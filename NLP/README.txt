There are four Python scripts included in this folder. At the top of every script is a series of global parameters which can be tweaked to alter performance.
 'Maximum.py' is the main script for Method 1: Maximum Normalised Frequency
 'Cumulative.py' is the main script for Method 2: Cumulative Normalised Frequency
 'Maximum_CV.py' is the cross-validation script for Method 1
 'Cumulative_CV.py' is the cross-validation script for Method 2
